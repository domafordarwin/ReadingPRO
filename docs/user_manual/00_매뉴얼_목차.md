## 역할별 주요 기능 요약

```
학생 (Student)
├── 대시보드 — 나의 학습 현황 한눈에 보기
├── 문해력 검사 — 진단 응시 (객관식 + 서술형)
├── 나의 진단 리포트 — 7개 섹션 종합 보고서
├── 발문 학습 — 3단계 (책문열기 → 이야기나누기 → 삶적용)
├── 나의 역량 — 발문 학습 진행도
├── 상담 신청 — 선생님께 상담 요청
└── 공지사항

학부모 (Parent)
├── 대시보드 — 자녀 학습 현황 종합
├── 자녀 관리 — 연결된 자녀 정보
├── 결과 리포트 — 자녀 종합 보고서 열람
├── 상담 신청 — 담당 선생님 상담
└── 학부모 포럼

진단교사 (Diagnostic Teacher)
├── 진단 관리
│   ├── 학교 담당자 관리
│   ├── 학생별 진단 배정 (개별/일괄)
│   ├── 학생답 관리 (엑셀 업로드/다운로드)
│   └── 진단 결과 리포트
├── 진단 분석
│   ├── 응시/채점 현황
│   ├── 학생별 피드백 (AI 자동 생성 + 배포)
│   └── 종합 결과 보고서 (AI 7개 섹션 + 배포)
├── 발문 학습
│   ├── 발문 모듈 관리 (CRUD)
│   ├── 발문 배정 관리 (개별/일괄)
│   └── 학생 세션 현황 (리뷰/채점/보고서)
└── 공지사항 및 상담
    ├── 공지사항 관리
    ├── 학생 상담 / 학부모 상담
    └── 상담 통계

학교관리자 (School Admin)
├── 대시보드 — 학교 통계
├── 학교 운영
│   ├── 학생/학급 관리 (CRUD + 일괄가져오기)
│   ├── 학부모 관리
│   ├── 진단 배포/운영 (배정/회수)
│   ├── 리포트 대시보드
│   ├── 공지사항 / 담당자 관리
│   └── 상담 통계 / 게시판 관리

문항개발자 (Researcher)
├── 대시보드 — 통계 + 빠른 액션
├── 콘텐츠 관리
│   ├── 평가 영역 (대분류 + 세부 지표)
│   ├── 모듈 관리 (지문 + 문항 묶음)
│   ├── 모듈 자동 생성 (PDF → AI 파싱)
│   ├── 지문 관리 (CRUD + AI 분석)
│   ├── 문항 관리 / 문항 등록 (MCQ + 서술형)
│   ├── 진단지 구성 (모듈 조합)
│   ├── 독서력 진단지
│   ├── 프롬프트 관리
│   └── 도서 관리

시스템관리자 (Admin)
├── 시스템 관리 — 실시간 성능 모니터링
├── 사용자 관리 — 전체 사용자 CRUD + 역할 변경
├── 역할 관리 — 7개 역할 확인 (읽기 전용)
├── 공지사항 / 공시사항
└── 에러 로그 — 에러 수집 + AI 분석
```

## 이미지 촬영 가이드

### 촬영 환경

1. **브라우저:** Chrome (최신 버전)
2. **해상도:** 1920×1080 (Full HD)
3. **테스트 계정:** 각 역할별 테스트 계정 사용

### 테스트 계정 정보

| 역할 | 이메일 | 비밀번호 |
|------|--------|---------|
| 학생 | `shinmyung_s-0001@shinmyung.edu` | `shinmyung.edu_$12#` |
| 학부모 | `shinmyung_p-0001@shinmyung.edu` | `shinmyung.edu_$12#` |
| 진단교사 | `teacher_diagnostic@ReadingPro.com` | `ReadingPro$12#` |
| 학교관리자 | (학교별 계정) | `{도메인}_$12#` |
| 문항개발자 | `researcher@ReadingPro.com` | `ReadingPro$12#` |
| 시스템관리자 | `admin@ReadingPro.com` | `ReadingPro$12#` |

### 촬영 팁

- **강조 표시**: Canva에서 빨간 테두리/화살표로 클릭 위치 표시
- **개인정보**: 학생 실명이 노출되지 않도록 테스트 데이터 사용
- **일관성**: 모든 스크린샷은 동일한 화면 비율(16:9)로 촬영
- **번호 표시**: 여러 영역을 설명할 때 1, 2, 3... 원형 번호로 표시
