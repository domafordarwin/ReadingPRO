## 2. 대시보드

### 2.1 학교 기본 정보

- 학교명, 소재지, 학교 유형이 표시됩니다.

### 2.2 학생 통계

| 통계 | 내용 |
|------|------|
| **총 학생 수** | 우리 학교에 등록된 전체 학생 |
| **학급 수** | 학년/반 수 |
| **진단 참여율** | 진단에 참여한 학생 비율 |

### 2.3 진단 결과 요약

- 학년별 평균 진단 결과가 차트로 표시됩니다.
- 리포트 생성/배포 현황을 확인할 수 있습니다.

> **[이미지 필요: IMG-SA-02]** 대시보드 통계 + 학년별 차트

## 3. 학생/학급 관리

**메뉴 위치:** 사이드바 > 학교 운영 > **학생/학급 관리**

> **[이미지 필요: IMG-SA-03]** 학생 목록 화면 — 검색, 편집, 삭제 버튼

### 3.1 학생 목록 조회

- **검색:** 이름 또는 학번으로 검색 가능
- **표시 정보:** 이름, 학년, 반, 이메일, 상태
- **페이지네이션:** 20명씩 표시

### 3.2 학생 정보 편집

1. 학생 옆의 **"편집"** 버튼을 클릭합니다.
2. 이름, 학년, 반 등을 수정합니다.
3. **"저장"** 버튼을 클릭합니다.

### 3.3 비밀번호 초기화

- 학생이 비밀번호를 잊었을 때 사용합니다.
- **"비밀번호 초기화"** 버튼을 클릭하면 초기 비밀번호로 재설정됩니다.
- 초기 비밀번호: `{학교도메인}_$12#`

> **[이미지 필요: IMG-SA-04]** 비밀번호 초기화 확인 대화상자

### 3.4 학생 삭제

- **"삭제"** 버튼을 클릭합니다.
- **주의:** 삭제 시 해당 학생의 모든 진단 데이터도 함께 삭제됩니다.

### 3.5 학생 일괄 가져오기

1. **"일괄 가져오기"** 버튼을 클릭합니다.
2. CSV 파일을 업로드합니다.
3. 업로드 결과(성공/실패 건수)를 확인합니다.

> **[이미지 필요: IMG-SA-05]** 학생 일괄 가져오기 — CSV 업로드 인터페이스

**CSV 형식:**

| 열 | 내용 | 예시 |
|----|------|------|
| A | 학생 이름 | 김철수 |
| B | 학년 | 3 |
| C | 반 | 2반 |

## 4. 학부모 관리

**메뉴 위치:** 사이드바 > 학교 운영 > **학부모 관리**

> **[이미지 필요: IMG-SA-06]** 학부모 목록 화면

### 4.1 학부모 목록

- 우리 학교 소속 학부모만 표시됩니다.
- 이름, 이메일, 연결된 자녀 정보가 표시됩니다.
- 20명씩 페이지네이션

### 4.2 학부모 정보 편집 / 비밀번호 초기화 / 삭제

학생 관리와 동일한 방식으로 편집, 비밀번호 초기화, 삭제가 가능합니다.

## 5. 진단 배포/운영

**메뉴 위치:** 사이드바 > 학교 운영 > **진단 배포/운영**

> **[이미지 필요: IMG-SA-07]** 진단 배포/운영 화면 — 진단지별 배정 현황

### 5.1 배정 현황 확인

- 우리 학교에 배정된 진단지 목록이 표시됩니다.
- 각 진단지별 학생 배정/완료 현황을 확인합니다.

### 5.2 개별 배정

1. 진단지를 선택합니다.
2. **"개별 배정"** 버튼을 클릭합니다.
3. 학생을 선택하고 배정합니다.

### 5.3 일괄 배정

1. **"일괄 배정"** 버튼을 클릭합니다.
2. 학년/반 또는 전체 학생을 선택합니다.
3. 배정을 확인하고 실행합니다.

> **[이미지 필요: IMG-SA-08]** 일괄 배정 인터페이스 — 학년/반 선택

### 5.4 배정 회수

- "진행 중" 또는 "배정됨" 상태인 배정만 회수 가능합니다.
- **"회수"** 버튼을 클릭합니다.

## 9. 자주 묻는 질문 (FAQ)

### Q. 학생이 비밀번호를 몰라요.
**A.** 학생/학급 관리에서 해당 학생의 **"비밀번호 초기화"**를 클릭하세요. 초기 비밀번호(`{학교도메인}_$12#`)로 재설정됩니다.

### Q. 학생을 일괄 등록하고 싶어요.
**A.** 학생/학급 관리에서 **"일괄 가져오기"** → CSV 파일 업로드로 한번에 등록할 수 있습니다.

### Q. 진단 배정을 취소하고 싶어요.
**A.** 진단 배포/운영에서 "배정됨" 또는 "진행 중" 상태인 배정만 **"회수"** 버튼으로 취소할 수 있습니다. "완료" 상태는 취소 불가합니다.

### Q. 다른 학교의 학생이 보여요.
**A.** 학교관리자는 자기 학교 소속 학생/학부모만 볼 수 있습니다. 다른 학교 데이터가 보인다면 관리자에게 문의하세요.
