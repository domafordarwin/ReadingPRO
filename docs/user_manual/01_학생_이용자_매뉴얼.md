## 1. 로그인

### 1.1 접속 방법

1. 웹 브라우저에서 ReadingPRO 주소로 접속합니다.
2. 메인 화면에서 **"로그인"** 버튼을 클릭합니다.

> **[이미지 필요: IMG-S-01]** 메인 페이지 스크린샷 — 로그인 버튼 위치를 빨간 테두리로 표시

### 1.2 계정 정보 입력

| 항목 | 입력 내용 | 예시 |
|------|----------|------|
| **이메일** | 학교에서 발급받은 학생 ID | `shinmyung_s-0001@shinmyung.edu` |
| **비밀번호** | 초기 비밀번호 (학교 도메인 기반) | `shinmyung.edu_$12#` |

3. 이메일과 비밀번호를 입력합니다.
4. **"로그인"** 버튼을 클릭합니다.
5. 처음 로그인 시 **비밀번호 변경 화면**으로 자동 이동합니다.

> **[이미지 필요: IMG-S-02]** 로그인 폼 스크린샷 — 이메일/비밀번호 필드, 로그인 버튼 표시

### 1.3 비밀번호 변경 (최초 1회)

처음 로그인하면 반드시 비밀번호를 변경해야 합니다.

1. **현재 비밀번호**: 초기 비밀번호를 입력합니다.
2. **새 비밀번호**: 원하는 비밀번호를 입력합니다 (8자 이상 권장).
3. **비밀번호 확인**: 새 비밀번호를 한 번 더 입력합니다.
4. **"변경하기"** 버튼을 클릭합니다.

> **[이미지 필요: IMG-S-03]** 비밀번호 변경 화면 스크린샷

### 1.4 로그인 실패 시

- 비밀번호를 **3회 연속 틀리면** 계정이 **1시간 동안 잠깁니다**.
- 잠긴 경우 담당 선생님에게 비밀번호 초기화를 요청하세요.

## 2. 대시보드

로그인 후 가장 먼저 보이는 화면입니다. 나의 학습 현황을 한눈에 확인할 수 있습니다.

> **[이미지 필요: IMG-S-04]** 학생 대시보드 전체 화면 스크린샷 — 각 영역에 번호 표시 (1~6)

### 2.1 상단 통계 카드

| 카드 | 내용 | 설명 |
|------|------|------|
| 완료한 진단 수 | 숫자 표시 | 지금까지 완료한 진단 검사 횟수 |
| 배정된 진단 | 숫자 표시 | 현재 응시해야 할 진단 목록 |

### 2.2 배정된 진단 목록

- 선생님이 나에게 배정한 진단 검사가 표시됩니다.
- **"시작하기"** 버튼을 클릭하면 바로 검사를 시작할 수 있습니다.

### 2.3 역량 요약

- 가장 최근 진단 결과를 바탕으로 나의 읽기 역량을 보여줍니다.
- **레이더 차트**: 9개 하위 지표별 점수를 시각적으로 표시합니다.

> **[이미지 필요: IMG-S-05]** 역량 요약 레이더 차트 영역 확대 스크린샷

### 2.4 최근 피드백

- 선생님이 작성해 주신 최근 피드백 6개가 표시됩니다.
- 클릭하면 상세 피드백을 확인할 수 있습니다.

### 2.5 공지사항

- 최근 공지사항 5개가 표시됩니다.

## 3. 문해력 검사 (진단 응시)

### 3.1 진단 목록 확인

왼쪽 메뉴에서 **"문해력 검사"**를 클릭합니다.

> **[이미지 필요: IMG-S-06]** 사이드바 메뉴 스크린샷 — "문해력 검사" 메뉴 위치 강조

화면에 다음 3가지 상태의 진단이 표시됩니다:

| 상태 | 의미 | 할 일 |
|------|------|-------|
| **배정됨** | 아직 시작하지 않은 진단 | "시작하기" 클릭 |
| **진행 중** | 시작했지만 아직 완료하지 않은 진단 | "이어서 하기" 클릭 |
| **완료** | 모든 문항에 응답을 마친 진단 | 결과 확인 가능 |

### 3.2 진단 응시하기

1. 배정된 진단에서 **"시작하기"** 버튼을 클릭합니다.
2. 지문(읽기 자료)을 꼼꼼히 읽습니다.
3. 문항에 답변합니다:

#### 객관식 문항
- 보기 중 하나를 선택합니다.
- 선택한 답은 자동 저장됩니다.

> **[이미지 필요: IMG-S-07]** 객관식 문항 응시 화면 — 지문 영역 + 보기 선택 영역

#### 서술형 문항
- 텍스트 입력란에 자유롭게 답변을 작성합니다.
- 작성한 답은 자동 저장됩니다.

> **[이미지 필요: IMG-S-08]** 서술형 문항 응시 화면 — 지문 영역 + 텍스트 입력 영역

### 3.3 답안 마킹 기능

- 나중에 다시 확인하고 싶은 문항에 **깃발 아이콘**을 클릭하여 마킹할 수 있습니다.
- 마킹한 문항은 표시되어 쉽게 찾을 수 있습니다.

### 3.4 제출하기

- 모든 문항에 답변을 완료한 후 **"제출"** 버튼을 클릭합니다.
- 제출 후에는 답변을 수정할 수 없으므로 신중하게 확인하세요.

> **[이미지 필요: IMG-S-09]** 제출 확인 대화상자 스크린샷

## 4. 나의 진단 리포트

### 4.1 리포트 확인하기

왼쪽 메뉴에서 **"나의 진단 리포트"**를 클릭합니다.

- 선생님이 **배포한 종합 보고서**만 이 페이지에 표시됩니다.
- 보고서가 1건이면 자동으로 상세 내용이 표시됩니다.
- 보고서가 2건 이상이면 목록에서 선택할 수 있습니다.

> **[이미지 필요: IMG-S-10]** 종합 보고서 상세 페이지 스크린샷 — 7개 섹션 목차 표시

### 4.2 보고서 구성 (7개 섹션)

| 섹션 | 내용 |
|------|------|
| 1. 기초 정보 | 학생 이름, 학년, 진단지, 응시일 |
| 2. 전체 요약 | 종합 분석 결과 요약 |
| 3. 영역별 분석 | 평가 영역(대분류)별 상세 분석 |
| 4. 세부 지표 분석 | 하위 지표(소분류)별 강점/약점 |
| 5. 읽기 성향 분석 | 독자 유형과 읽기 패턴 분석 |
| 6. 학습 권장 사항 | AI가 분석한 맞춤 학습 추천 |
| 7. 종합 소견 | 선생님의 종합 의견 |

### 4.3 시도별 상세 결과

- 각 진단 시도의 문항별 응답과 점수를 확인할 수 있습니다.
- 객관식: 선택한 답과 정답 비교
- 서술형: 내 답변과 채점 기준(루브릭) 점수

> **[이미지 필요: IMG-S-11]** 시도별 상세 결과 화면 — 객관식/서술형 분리 표시

## 5. 발문 학습

### 5.1 발문 학습이란?

글을 읽고 스스로 질문을 만들어 보는 학습 활동입니다. 3단계로 구성됩니다:

| 단계 | 이름 | 활동 |
|------|------|------|
| **1단계** | 책문열기 | 글을 읽고 궁금한 점에 대한 질문 만들기 |
| **2단계** | 이야기나누기 | AI와 대화하며 생각 나누기 |
| **3단계** | 삶적용 | 배운 내용을 일상에 적용하는 글쓰기 |

### 5.2 발문 학습 시작하기

1. 왼쪽 메뉴에서 **"발문 학습"**을 클릭합니다.
2. 배정된 발문 모듈 목록이 표시됩니다.
3. 학습할 모듈의 **"시작하기"** 버튼을 클릭합니다.

> **[이미지 필요: IMG-S-12]** 발문 학습 모듈 목록 화면 — 모듈 카드와 시작 버튼

### 5.3 1단계: 책문열기

1. 제시된 읽기 지문을 주의 깊게 읽습니다.
2. 지문에 대한 **질문을 직접 작성**합니다.
3. AI가 질문의 질을 평가하고 점수를 매깁니다.
4. 필요 시 **스캐폴딩(도움말)**이 제공됩니다.

> **[이미지 필요: IMG-S-13]** 책문열기 단계 화면 — 지문 영역 + 질문 입력란 + AI 피드백

### 5.4 2단계: 이야기나누기

1. AI와 **토론 형식으로 대화**합니다.
2. AI가 추가 질문을 하면 자신의 생각을 자유롭게 답합니다.
3. 대화 내용은 자동 저장됩니다.

> **[이미지 필요: IMG-S-14]** 이야기나누기 단계 화면 — 채팅 형식 대화 인터페이스

### 5.5 3단계: 삶적용

1. 학습 내용을 바탕으로 **논술문(에세이)**을 작성합니다.
2. 작성 완료 후 제출합니다.
3. AI가 에세이를 평가하고 피드백을 제공합니다.

> **[이미지 필요: IMG-S-15]** 삶적용 단계 화면 — 에세이 작성 에디터

### 5.6 학습 수준 시스템

- 학습 수준은 4단계로 나뉩니다:

| 수준 | 대상 | 난이도 |
|------|------|--------|
| elementary_low | 초등 저학년 | 가장 쉬움 |
| elementary_high | 초등 고학년 | 쉬움 |
| middle | 중학생 | 보통 |
| high | 고등학생 | 어려움 |

- **잘하면 수준이 자동으로 올라갑니다!** (마스터리 90% 이상 달성 시)
- 어려우면 **스캐폴딩(도움말)**이 자동 제공됩니다.

## 10. 자주 묻는 질문 (FAQ)

### Q. 비밀번호를 잊어버렸어요.
**A.** 담당 선생님에게 비밀번호 초기화를 요청하세요. 초기화 후 초기 비밀번호로 로그인하고 새 비밀번호를 설정합니다.

### Q. 진단 검사 도중에 브라우저를 닫아도 되나요?
**A.** 네, 답변은 자동 저장됩니다. 다시 로그인하면 "진행 중" 상태에서 이어서 할 수 있습니다.

### Q. 제출 후 답변을 수정할 수 있나요?
**A.** 아니요, 제출 후에는 수정할 수 없습니다. 제출 전에 꼼꼼히 확인하세요.

### Q. 나의 진단 리포트가 보이지 않아요.
**A.** 선생님이 리포트를 배포해야 표시됩니다. 선생님에게 문의하세요.

### Q. 발문 학습에서 수준이 올라가지 않아요.
**A.** 마스터리 점수 90% 이상, 스캐폴딩 수준 0일 때 자동으로 올라갑니다. 꾸준히 학습하세요.

### Q. 계정이 잠겼어요.
**A.** 비밀번호를 3회 연속 틀리면 1시간 동안 잠깁니다. 1시간 후 다시 시도하거나, 선생님에게 초기화를 요청하세요.
