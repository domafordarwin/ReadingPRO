## 1. 로그인

### 1.1 접속 방법

1. 웹 브라우저에서 ReadingPRO 주소로 접속합니다.
2. 메인 화면에서 **"로그인"** 버튼을 클릭합니다.

### 1.2 계정 정보 입력

| 항목 | 입력 내용 | 예시 |
|------|----------|------|
| **이메일** | 학교에서 발급받은 학부모 ID | `shinmyung_p-0001@shinmyung.edu` |
| **비밀번호** | 초기 비밀번호 (학교 도메인 기반) | `shinmyung.edu_$12#` |

> **[이미지 필요: IMG-P-01]** 로그인 화면 — 이메일/비밀번호 입력 필드

### 1.3 비밀번호 변경 (최초 1회)

처음 로그인 시 **비밀번호 변경이 필수**입니다. 새 비밀번호를 입력하고 변경하기를 클릭합니다.

### 1.4 로그인 실패 시

- 비밀번호 **3회 연속 오류** 시 계정이 **1시간 잠금**됩니다.
- 잠금 시 담당 선생님에게 비밀번호 초기화를 요청하세요.

## 2. 대시보드

로그인 후 가장 먼저 보이는 화면입니다. 자녀의 학습 현황을 종합적으로 확인할 수 있습니다.

> **[이미지 필요: IMG-P-02]** 학부모 대시보드 전체 화면 — 각 영역에 번호 표시

### 2.1 상단 요약 카드

| 카드 | 내용 |
|------|------|
| **자녀 수** | 연결된 자녀 수 |
| **활동 중인 자녀** | 현재 진단에 참여하고 있는 자녀 수 |
| **총 진단 수** | 자녀들이 완료한 진단 총 횟수 |
| **평균 점수** | 전체 자녀의 평균 진단 점수 |
| **대기 중 상담** | 답변을 기다리는 상담 수 |

### 2.2 최근 활동

- 자녀의 최근 평가 활동과 상담 내역이 시간순으로 표시됩니다.

### 2.3 자녀별 진행 현황

- 각 자녀의 진단 진행 상황과 점수 추세를 확인할 수 있습니다.
- 그래프로 점수 변화를 시각적으로 보여줍니다.

> **[이미지 필요: IMG-P-03]** 자녀별 진행 현황 — 추세 그래프 영역 확대

## 3. 자녀 관리

### 3.1 자녀 목록 확인

왼쪽 메뉴에서 **"자녀 관리"**를 클릭합니다.

> **[이미지 필요: IMG-P-04]** 사이드바 메뉴 — "자녀 관리" 위치 강조

- 연결된 자녀 목록이 표시됩니다.
- 각 자녀의 이름, 학년, 반, 최근 진단 상태를 확인할 수 있습니다.

### 3.2 자녀별 상세 정보

자녀 이름을 클릭하면 해당 자녀의 상세 정보를 볼 수 있습니다:

- 기본 정보 (이름, 학년, 반)
- 진단 참여 이력
- 최근 점수

> **[이미지 필요: IMG-P-05]** 자녀 상세 정보 화면

## 4. 결과 리포트

### 4.1 리포트 확인하기

왼쪽 메뉴에서 **"결과 리포트"**를 클릭합니다.

- 첫 번째 자녀(또는 선택한 자녀)의 **최신 종합 보고서**가 자동으로 표시됩니다.
- 자녀가 여러 명이면 상단에서 자녀를 선택할 수 있습니다.

> **[이미지 필요: IMG-P-06]** 결과 리포트 페이지 — 자녀 선택 + 보고서 영역

### 4.2 보고서 내용 이해하기

종합 보고서는 7개 섹션으로 구성됩니다:

| 섹션 | 학부모가 주목할 점 |
|------|------------------|
| **1. 기초 정보** | 자녀의 진단 일시, 진단지 종류 확인 |
| **2. 전체 요약** | 전반적인 읽기 능력 수준 파악 |
| **3. 영역별 분석** | 어떤 영역이 강하고 약한지 확인 |
| **4. 세부 지표 분석** | 구체적인 강점과 약점 파악 |
| **5. 읽기 성향 분석** | 자녀의 독서 유형과 패턴 이해 |
| **6. 학습 권장 사항** | **가정에서 할 수 있는 학습 활동** 확인 |
| **7. 종합 소견** | 선생님의 전문적인 의견 확인 |

> **[이미지 필요: IMG-P-07]** 보고서 상세 — 학습 권장 사항 섹션 확대

### 4.3 전체 리포트 목록

- "전체 리포트 보기"를 클릭하면 자녀의 모든 진단 기록을 확인할 수 있습니다.
- 과거 결과와 비교하여 성장 추이를 확인하세요.

### 4.4 응답 상세 보기

- 각 시도를 클릭하면 문항별 응답 내용을 확인할 수 있습니다.
- 객관식: 자녀가 선택한 답과 정답
- 서술형: 자녀가 작성한 답변과 채점 결과

## 5. 상담 신청

### 5.1 상담 신청하기

1. 대시보드 또는 자녀 정보 페이지에서 **"상담 신청"** 버튼을 클릭합니다.
2. 상담 내용을 작성합니다:
   - **상담 유형** 선택 (학습 관련, 진로 상담 등)
   - **상담 내용** 상세 기술
3. **"제출"** 버튼을 클릭합니다.

> **[이미지 필요: IMG-P-08]** 상담 신청 폼 화면

### 5.2 상담 진행 확인

| 상태 | 의미 |
|------|------|
| **대기 중** | 선생님이 아직 확인하지 않은 상태 |
| **승인** | 선생님이 상담을 승인한 상태 |
| **거절** | 선생님이 상담을 거절한 상태 (사유 확인 가능) |
| **완료** | 상담이 완료된 상태 |

## 6. 학부모 포럼

### 6.1 포럼 이용하기

왼쪽 메뉴에서 **"학부모 포럼"**을 클릭합니다.

- 다른 학부모와 정보를 공유할 수 있는 게시판입니다.
- 게시물을 읽고 댓글을 남길 수 있습니다.

> **[이미지 필요: IMG-P-09]** 학부모 포럼 목록 화면

### 6.2 댓글 작성

1. 게시물 하단의 댓글 입력란에 내용을 작성합니다.
2. **"등록"** 버튼을 클릭합니다.

### 6.3 내 댓글 삭제

- 내가 작성한 댓글만 삭제할 수 있습니다.
- 댓글 옆의 **"삭제"** 버튼을 클릭합니다.

## 7. 내 정보 관리

### 7.1 내 정보 확인

화면 상단의 **프로필 아이콘**을 클릭하면 내 정보를 확인할 수 있습니다.

### 7.2 비밀번호 변경

1. 프로필 메뉴에서 **"비밀번호 변경"**을 선택합니다.
2. 현재 비밀번호와 새 비밀번호를 입력합니다.
3. **"변경하기"** 버튼을 클릭합니다.

### 7.3 로그아웃

- 화면 하단 사이드바의 **"로그아웃"** 버튼을 클릭합니다.

## 8. 자주 묻는 질문 (FAQ)

### Q. 자녀의 진단 결과가 보이지 않아요.
**A.** 선생님이 종합 보고서를 배포해야 결과를 볼 수 있습니다. 담당 선생님에게 문의하세요.

### Q. 자녀가 여러 명인데 한 명만 보여요.
**A.** 상단의 자녀 선택 메뉴에서 다른 자녀를 선택할 수 있습니다.

### Q. 상담 신청 후 언제 답변을 받을 수 있나요?
**A.** 선생님의 업무 상황에 따라 다르지만, 보통 2~3일 이내에 답변됩니다. 대시보드에서 상담 상태를 확인하세요.

### Q. 비밀번호를 잊어버렸어요.
**A.** 담당 선생님 또는 학교 관리자에게 비밀번호 초기화를 요청하세요.

### Q. 자녀의 점수가 낮게 나왔어요. 어떻게 해야 하나요?
**A.** 보고서의 **"학습 권장 사항"** 섹션을 참고하세요. AI가 분석한 맞춤 학습 방법이 제시되어 있습니다. 추가 상담이 필요하면 상담 신청을 이용하세요.
