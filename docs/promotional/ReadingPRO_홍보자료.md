## [1페이지] 표지

# ReadingPRO
### AI 기반 읽기 능력 진단 및 성장 플랫폼

> "읽기 능력의 정확한 진단부터 개인 맞춤 성장까지,
> AI가 함께하는 독서 교육의 새로운 기준"

**대상:** 초등학교 1학년 ~ 고등학교
**개발:** 읽기 교육 연구팀
**기술:** AI(GPT-4) + 웹 플랫폼

---

## [2페이지] 왜 ReadingPRO인가?

### 읽기 교육의 현실

| 현재의 문제 | ReadingPRO의 해결 |
|------------|-----------------|
| 학생 30명의 읽기 수준을 교사 1명이 파악하기 어려움 | AI가 학생별 읽기 역량을 **자동 진단** |
| 진단 결과가 점수로만 제공되어 활용도 낮음 | **7개 영역 종합 분석 보고서** 자동 생성 |
| 읽기 교육이 일회성으로 끝남 | **3단계 발문 학습**으로 지속적 사고력 신장 |
| 학부모가 자녀 읽기 수준을 모름 | **학부모 대시보드**에서 실시간 성장 추적 |
| 학교 전체의 읽기 수준 현황 파악 불가 | **학교 관리자 대시보드**로 학년별 현황 한눈에 |

### 핵심 가치

**정확한 진단** - 전문 연구진이 개발한 평가 문항 + AI 자동 채점
**개인 맞춤** - 학생별 수준에 맞는 스캐폴딩과 피드백
**교사 업무 경감** - AI가 초안을 작성하고 교사가 검토/수정
**데이터 기반 의사결정** - 학교/학년/학급별 읽기 현황 대시보드

### 3단계 학습 구조

| 단계 | 이름 | 목적 | 활동 |
|------|------|------|------|
| 1단계 | **책문열기** | 텍스트 이해 | 지문 읽기, 핵심 파악 질문 |
| 2단계 | **이야기나누기** | 비판적 사고 | 분석, 추론, 평가 질문 |
| 3단계 | **삶적용** | 창의적 적용 | 실생활 연결, 대안 제시 |

### 차별점

- **AI 즉각 피드백**: 제출 즉시 AI가 강점/발전점 분석 (교사 기다릴 필요 없음)
- **수준별 자동 조절**: 4개 수준 (초저/초고/중등/고등) + 스캐폴딩 자동 조정
- **자동 수준 향상**: 숙달도 90% 이상 달성 시 다음 수준으로 자동 진급
- **교사 이중 검토**: AI 평가 후 교사가 최종 검토/수정 가능

## [10페이지] 기대 효과 및 문의

### 기대 효과

**학생**
- 자신의 읽기 수준과 성장 과정을 시각적으로 확인
- AI 즉각 피드백으로 자기주도적 학습 가능
- 수준별 맞춤 발문으로 사고력 단계적 신장

**교사**
- AI 자동 채점/피드백으로 **업무 시간 70% 이상 절감**
- 데이터 기반 학생 지도 및 학부모 상담
- 전문 연구진 개발 문항으로 평가의 전문성 확보

**학교**
- 학교 전체 읽기 교육 현황 데이터 확보
- 학년별/학급별 비교 분석으로 교육 정책 수립 근거 마련
- 학부모 소통 강화 (보고서 공유, 상담 자료)
